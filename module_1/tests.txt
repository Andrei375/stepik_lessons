Регистрация пользователя
Предусловия: открыта страница регистрации
Шаги:
Шаг 1 Заполнить поле "Адрес электронной почты" (kosmos09@tut.by)
-- Проверка ввода адреса без символа @
    Ожидаемый результат - Под полем "Адрес электронной почты" появляется информационное окно
     с предупреждением о пропуске символа @
Шаг 2 Заполнить поле "Пароль" (iXXeu22DPA4jv9Z)
-- Проверка краткого пароля - ввод пароля менее 9 символов
-- Ожидаемый результат - Под полем "Повторите пароль" появится надпись
    "Введённый пароль слишком короткий. Он должен содержать как минимум 9 символов."
Шаг 3 Нажать кнопку "Зарегистрироваться"
-- Ожидаемый результат - Открывается страница пользователя

Вход пользователя в свой аккаунт
Предусловия: открыта страница входа в аккаунт
Шаг 1 Заполнить поле "Адрес электронной почты" (kosmos09@tut.by)
-- Проверка ввода адреса без символа @
    Ожидаемый результат - Под полем "Адрес электронной почты" появляется информационное окно
     с предупреждением о пропуске символа @
Шаг 2 Заполнить поле "Пароль" (iXXeu22DPA4jv9Z)
-- Проверка некорректного пароля - ввод некорректного пароля
   Ожидаемый результат - Под полем "Повторите пароль" появится надпись
    "Введённый пароль слишком короткий. Он должен содержать как минимум 9 символов."
-- Проверка входа при вводе только адреса электронной почты без ввода пароля
   Ожидаемый результат - Появляется информационное окно "Заполните это поле" под полем "Пароль"
Шаг 3 Нажать кнопку "Войти"
-- Ожидаемый результат - Открывается страница пользователя
