Оформление заказа
Предусловия: открыта страница "Корзина", в корзину добавлено несколько товаров из разных групп
Шаг 1 Кликнуть кнопку "Перейти к оформлению" внизу страницы корзины
-- Проверка кликнуть кнопку "У меня есть код скидки"
    Ожидаемый результат - откроется поле ввода кода скидки и появится кнопка "Применить"
Шаг 2 Заполнить обязательные поля ввода адреса доставки
    Имя "Николай"
    Фамилия "Штирлиц"
    Первая строка адреса "ул. Василия Быкова 19/80"
    Город "Минск"
    Почтовый индекс "220000"
    Страна - выбрать из выпадающего списка "Belarus"
-- Проверка - оставить незаполненным обязательное поле ввода - Первая строка адреса
   Ожидаемый результат - Появляется информационное окно "Заполните это поле" под незаполненным полем
-- Проверка заполнить только необязательные поля ввода
   Ожидаемый результат - Появляется информационное окно "Заполните это поле" под первым незаполненным полем
Шаг 3 Нажать кнопку "Продолжить"
   Ожидаемый результат - Открывается страница параметров платежа
Шаг 4 Нажать кнопку "Продолжить"
   Ожидаемый результат - Открывается страница "Предпросмотр платежа"
Шаг 5 Нажать кнопку "Разместить заказ"
   Ожидаемый результат - Открывается страница "Подтверждение заказа"
-- Проверка - Кликнуть кнопку "Напечатать эту страницу"
   Ожидаемый оезультат - откроется окно печати принтера
-- Проверка - Кликнуть кнопку "Продолжить покупки"
   Ожидаемый результат - откроется страница каталога товаров
